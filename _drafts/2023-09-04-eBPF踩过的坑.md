## 内存对齐padding问题：  

编译后加载，verify报错:  
`invalid indirect read from stack R2 off -40+10 size 12`  
```
struct typeA {
    __u32 itemA;
    __u32 itemB;
    __u16 itemC;
};

struct {
    __uint(type, BPF_MAP_TYPE_HASH);
    __uint(max_entries, 10000);
    __type(key, struct typeA);
    __type(value, __u32);
} mapA SEC(".maps");
```
解决办法，手动填充padding：  
```
struct typeA {
    __u32 itemA;
    __u32 itemB;
    __u16 itemC;
    __u16 padding;
};
```
